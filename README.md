# Pixel Agents - IntelliJ Plugin

IntelliJ IDEA plugin for Pixel Agents.

Pixel art office visualization for AI coding agents. Watch Claude Code agents work as animated characters in a pixel art office.

## Credits

This plugin is based on [Pixel Agents](https://github.com/pablodelucca/pixel-agents) by pablodelucca (VS Code extension), adapted for the IntelliJ platform.
